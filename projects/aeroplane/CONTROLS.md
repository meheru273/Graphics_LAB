# 3D Airplane Simulator - Keyboard Controls

## Movement (Flight Simulator)
| Key | Action |
|-----|--------|
| W | Move forward |
| S | Move backward |
| A | Move left |
| D | Move right |
| E | Move up (increase altitude) |
| R | Move down (decrease altitude) |

## Rotation
| Key | Action |
|-----|--------|
| X | Pitch (nose up/down) |
| Y | Yaw (turn left/right) |
| Z | Roll (bank left/right) |

## Airplane Features
| Key | Action |
|-----|--------|
| G | Toggle engine / rotating fan |
| O | Open / close door |
| P | Open / close window shades |
| T | Takeoff sequence |
| U | Landing sequence |

## Lighting
| Key | Action |
|-----|--------|
| L | Toggle all lights on/off |
| 1 | Toggle directional light (sun) |
| 2 | Toggle point lights |
| 3 | Toggle spot light |
| 5 | Toggle ambient component |
| 6 | Toggle diffuse component |
| 7 | Toggle specular component |

## Camera
| Key | Action |
|-----|--------|
| C | Cycle camera mode (Follow → Bird's Eye → Free → Inside Cabin) |
| F | Orbit camera around plane |
| 4 | Toggle 4-viewport split view |
| Arrow Keys | Move free camera |
| Mouse | Look around (free cam mode) |
| Scroll | Zoom in/out |
| ESC | Exit |
